Sobre Mim

Desenvolvedor Full Stack Especializado em IA e Automação

Desenvolvedor Full Stack com foco em Java e Python, especializado no desenvolvimento de soluções completas que envolvem back-end robusto (REST APIs, automações, machine learning) e front-end intuitivo (Vue.js e React Native). Com mais de 3 anos de experiência prática em versionamento Git e frameworks modernos.

Atualmente trabalho como Software Developer na VJBots, onde desenvolvo soluções avançadas incluindo modelos de CNN (Convolutional Neural Network) para classificação de carcaças bovinas utilizando PyTorch, TensorFlow e Keras, além de implementações com YOLOv8 (Ultralytics) para identificação de humanos e objetos. Minha expertise inclui o desenvolvimento completo desde o levantamento de requisitos até o deploy em produção.

No desenvolvimento back-end, tenho experiência sólida com JPA, JSF, PrimeFaces, JSP, Java Servlets, Hibernate, criação de scrapers e automações via Selenium e Playwright. Também trabalho extensivamente com análise de dados usando Pandas e frameworks de Deep Learning como PyTorch, TensorFlow e Scikit-learn.

Sou movido por desafios reais, com foco em código limpo, testes automatizados e aprendizado contínuo. Valorizo a colaboração em equipe, organização e desenvolvimento de soluções escaláveis e bem estruturadas.
